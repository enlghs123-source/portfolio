모바일 뱅킹 앱 전면 리디자인

복잡하게 얽힌 정보 구조와 송금 플로우를 사용자 관점에서 다시 설계해, 핵심 작업을 더 빠르고 명확하게 만든 프로젝트입니다.

프로젝트 개요

7년차 사용자들이 가장 자주 쓰는 ‘송금’과 ‘조회’ 경험이 누적된 기능 추가로 복잡해졌습니다. 본 프로젝트는 정보 구조(IA)를 처음부터 재정의하고, 핵심 플로우를 3단계로 압축하며, 전 화면 접근성(WCAG AA)을 충족하는 것을 목표로 진행했습니다.

02 — Process

어떻게 해결했나

  1. 01

    리서치

    사용자 인터뷰 12명 + 행동 로그 분석으로 이탈 구간을 정의.

  2. 02

    IA 재설계

    기능 중심 구조 → 작업 중심 구조로 정보 구조를 재정의.

  3. 03

    프로토타입·테스트

    ProtoPie로 인터랙션 검증, Maze 원격 사용성 테스트 2라운드.

  4. 04

    디자인 시스템

    컴포넌트·토큰을 정비해 개발 핸드오프와 일관성을 확보.
